# Mode Inline Chat de GitHub Copilot

Le mode Inline Chat permet d’interagir directement avec l’éditeur ou le terminal de Visual Studio Code pour obtenir des suggestions, des explications ou de l’aide contextuelle, sans quitter le flux de travail. Il se distingue par son intégration contextuelle et sa capacité à proposer des modifications ou des commandes adaptées à la sélection ou à la position du curseur.

## Usages principaux

- Génération ou modification de code à la volée dans l’éditeur
- Explications ciblées sur une portion de code sélectionnée
- Suggestions de commandes shell ou d’actions dans le terminal intégré
- Refactorisation rapide ou ajout de gestion d’erreurs

> Pour des questions générales ou des explications sans modification de code, consulter le mode [Ask](./mode-ask.md). Pour toute demande de modification, de documentation ou de transformation de code, consulter les modes [Edit](./mode-edit.md) et [Agent](./mode-agent.md).
